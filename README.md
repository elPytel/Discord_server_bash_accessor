# Discord server bash accessor

## Configuration
Create a file named `config.json` in the root directory of the project and fill it with the following content:
``` json
{
    "API_TOKEN": "YOUR_API_TOKEN",
    "CHANNEL_ID": ,
    "SERVER_ID": ,
    "CATEGORY_ID": 
}
```

## pip dependencies
``` bash
pip install -r requirements.txt
```
